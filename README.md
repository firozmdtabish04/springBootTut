## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/springboot-learning.git
```

### Navigate

```bash
cd springboot-learning
```

### Configure Database

Update the `application.properties` file.

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/springboot_learning
spring.datasource.username=root
spring.datasource.password=your_password
```

### Run

```bash
mvn spring-boot:run
```

or

Run the project from Spring Tool Suite (STS).
